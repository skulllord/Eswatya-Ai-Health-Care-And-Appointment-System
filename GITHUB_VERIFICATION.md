### 2. Verify Model Files
```bash

# Should show files with sizes
```

### 3. Test Model Loading
```bash
python -c "import pickle; model = pickle.load(open('model.pkl', 'rb')); print(f'Success! {len(model.classes_)} diseases loaded')"
# Output: Success! 677 diseases loaded
```

### 4. Run Application
```bash
# Backend
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python init_db.py
python main.py

# Frontend (new terminal)
cd frontend
npm install
npm run dev
```

### 5. Access Application

- Should see landing page
